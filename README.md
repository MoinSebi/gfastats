# gfastats
#### Description 
Small tool for basic graph statistics using GFA format.   
  


####Installation: 

**Git**  
```
git clone https://github.com/MoinSebi/gfastats  
cd gfastats   
cargo build --release  
./target/release/gfastats  
```

**Cargo** 
```
argo  
daskdajd 
daskdaj
```

#### Usage
```
Usage:
  ./gfastats [OPTIONS]

GFA statistics

Optional arguments:
  -h,--help             Show this help message and exit
  -i,--input INPUT      Input data

```
